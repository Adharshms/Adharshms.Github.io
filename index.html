HTML Lists Example

Ordered List (Numbered)

Here is an example of an ordered list:

  1. First item
  2. Second item
  3. Third item
<ol>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ol>

Unordered List (Bulleted)

Here is an example of an unordered list:

  • First item
  • Second item
  • Third item
<ul>
    <li>First item</li>
    <li>Second item</li>
    <li>Third item</li>
</ul>
